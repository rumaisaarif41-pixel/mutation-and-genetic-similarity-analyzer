# mutation-and-genetic-similarity-analyzer:
Mutation and Genetic Similarity Analyzer lightweight Python tool designed to compare two DNA sequences, identify specific mutation points, and calculate the overall genetic similarity percentage.
##  Features:
Mutation Mapping: Identifies the exact index where sequences differ.
Similarity Scoring: Calculates the percentage match between two sequences
Clean Output: Provides clear terminal feedback for each detected mutation.
### How to Use:
Ensure you have Python 3.x installed.
Open mutation_and_similarity_analyser.py.
Update the dna_a and dna_b variables with your sequences
